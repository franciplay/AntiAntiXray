## Usage
To scan nearby blocks in selected radius, press `G` ingame.

## Installing
1. Download the newest version from the Releases tab
2. Put it into your mods folder in your fabric installation
3. Enjoy!

## NOTE

This is a Fabric only mod (for now) and it requires the Fabric api.
